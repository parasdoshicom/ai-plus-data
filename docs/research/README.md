# Research Guide

This folder holds source-backed research used to sharpen the repo's point of view. It is more memo-like than the core playbooks.

## Files

- [`ai-productivity-executive-summary.md`](./ai-productivity-executive-summary.md): short version of the research conclusions
- [`ai-productivity-for-top-data-teams.md`](./ai-productivity-for-top-data-teams.md): full operator memo with patterns, risks, and source links

- [`ai-productivity-repo-score.md`](./ai-productivity-repo-score.md): repo-specific scoring and editorial assessment

## Best Reading Order

1. Start with [`ai-productivity-executive-summary.md`](./ai-productivity-executive-summary.md).
2. Read [`ai-productivity-for-top-data-teams.md`](./ai-productivity-for-top-data-teams.md) for the full argument.
3. Use [`ai-productivity-source-log.md`](./ai-productivity-source-log.md) if you want the underlying source trail.
