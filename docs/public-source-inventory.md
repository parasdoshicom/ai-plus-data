## Internal Source Handling

Private internal source material was used only as inspiration for generalized patterns and was not copied directly into this repo.

Examples of source categories that informed the rewrite:

- AI workflow notes
- data team playbooks
- operator checklists
- public-style README structures from AI tooling projects

These were converted into public-safe artifacts and filtered through the publication rules in [`docs/repo-privacy-policy.md`](./repo-privacy-policy.md) and [`docs/publication-safety-checklist.md`](./publication-safety-checklist.md).

## Exclusions

This repo intentionally excludes:

- private meeting notes
- company-specific strategy documents
- internal SQL and warehouse identifiers
- compensation, performance, or HR content
- customer or employee data
- internal URLs, doc IDs, or dashboard links
- private Telegram or backend operating configuration
